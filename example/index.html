About

Stylizing a website can be a difficult task, right? Ozone is an ultra-simple style sheet that provides many tools to upgrade a website build process.

Simplicity

The ozone classes' names are short and easy to learn. Spend your time building the website and not learning how to use a complicated tool.

Design

Following a minimalist concept, Ozone provides cleanly designed elements to your website. Also, dark mode is always welcome.

Documentation

Where to find the docs?

To see the current documentation, visit Ozone's Github official page. You can also contribute to this project by reporting an issue or forking it. Briefly, the easiest way to implement Ozone in your code is to put the ozone.css file in your project and call it in the HTML file.
